# Launch Instructions

To view your 3D Particle App locally, you will need to serve the `index.html` file using a local web server. You can do this easily if you have Python or Node.js installed.

### Option 1: Using Python (Recommended)
If you have Python installed, open your terminal in this directory and run:
```bash
python -m http.server 8000
```
Then, click the link below to launch the app:
[http://localhost:8000](http://localhost:8000)

### Option 2: Using Node.js
If you have Node.js installed, you can use `npx` to serve the directory:
```bash
npx serve
```
Then, click the link below to launch the app:
[http://localhost:3000](http://localhost:3000)
